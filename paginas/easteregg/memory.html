  1. O jogo consiste em um conjunto de cartas viradas para baixo em uma mesa.
  2. Cada carta tem um símbolo em seu verso.
  3. O jogador pode virar duas cartas por vez, tentando encontrar pares correspondentes.
  4. Se as duas cartas viradas pelo jogador formarem um par correspondente, elas são removidas do jogo.
  5. Se as duas cartas não formarem um par correspondente, elas são viradas novamente para baixo.
  6. O tempo continua a ser contado enquanto o jogador faz suas jogadas.
  7. O jogo termina quando o jogador encontrar todos os pares.
  8. O objetivo é acertar todos os pares no menor tempo possível.
